# Scala类和对象

| 类型    | 描述                                                      |
| ------- | --------------------------------------------------------- |
| Null    | Trait，其唯一实例是null，是AnyRef的子类，不是AnyVal的子类 |
| Nothing | Trait，所有类型的子类（包括AnyRef和AnyVal），没有实例     |
| None    | Option的两个子类之一，另一个是Some，用于安全的函数返回值  |
| Unit    | 无返回值的函数的类型，和Java的void对应                    |
| Nil     | 长度为0的List                                             |
